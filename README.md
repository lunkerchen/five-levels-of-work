# Five Levels of Work

A decision and collaboration framework for high-agency teams and AI agents.

**Core thesis:** Level 4 is the baseline expectation; Level 5 is the goal.

| Level | Behavior | One-liner |
|-------|----------|-----------|
| L1 | "There's a problem." | Pure notification |
| L2 | "There's a problem, I found some causes." | Notification + diagnosis |
| L3 | "Here's the problem, some possible causes, and some solutions." | Diagnosis + options |
| L4 | "Here's the problem, what I think caused it, and which solution I recommend." | Diagnosis + recommendation |
| L5 | "I identified a problem, diagnosed it, fixed it, and I'm keeping you in the loop." | Autonomous resolution |

See [SKILL.md](SKILL.md) for domain-specific mappings and usage details.

# 五層工作等級

高主動性團隊與 AI Agent 的決策協作框架。

**核心命題：L4 是門檻，L5 是目標。**

| 等級 | 行為 | 一句話 |
|------|------|--------|
| L1 | 「有問題。」 | 純通報 |
| L2 | 「有問題，我找到一些原因。」 | 通報+診斷 |
| L3 | 「問題是 X，原因可能是 A/B，解法有 C/D。」 | 診斷+方案選項 |
| L4 | 「問題是 X，我認為主因是 A，建議解法 C。」 | 診斷+推薦 |
| L5 | 「我發現問題→診斷→修復→順便跟你說一聲。」 | 自主解決+知會 |

詳見 [SKILL.md](SKILL.md) 中各領域的對照表與使用說明。
